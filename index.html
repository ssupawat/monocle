How it works

1. Add blocks. Type statements inside, like p → q.

2. Lasso-select premise blocks (drag on empty canvas).

3. Click a block to set it as the conclusion. The selected premises now entail it.

Valid means: no case where all premises are true and the conclusion is false.

Trivial means: the conclusion is already one of the premises. Logically valid, but adds no new information.

Operators: ¬ ∧ ∨ → ↔, or type ! & | -> <->. Variables are letters like p, q.

A concluded block can itself be a premise in the next step. Chain steps to build longer arguments.
