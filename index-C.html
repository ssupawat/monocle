How it works

Block = a statement. Type any proposition inside using ¬ ∧ ∨ → ↔ (or ! & | -> <->) and variables like p.

∧ wire joins a block into the premise cluster (all must hold) — shown as a dashed halo around the group.

⊨ wire points from that whole cluster (the halo) to any proposition: “the premises ⊨ this”. That target is derived and can itself be ∧-linked into a later step.

Each ⊨ wire is its own checkable step. Valid ⟺ no row where all premises true and target false. Chain steps to build a longer argument.
